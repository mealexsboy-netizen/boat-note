# **App Name**: Mind Canvas

## Core Features:

- Infinite Canvas Interface: A borderless, free-form digital canvas where users can place notes anywhere, supporting limitless expansion.
- Freeform Text Note Creation: Ability to tap anywhere on the canvas to create a new text note and directly type content without constraints.
- Intuitive Note Manipulation: Easily drag and drop notes to reposition them, and tap existing notes to edit their text content.
- Zoom and Pan Navigation: Smooth navigation across the expansive canvas with pinch-to-zoom and drag-to-pan functionalities.
- Idea Expansion Tool: A generative AI tool that suggests expanded ideas or related concepts based on user input, creating new notes on the canvas.
- Automatic Canvas State Persistence: Automatically save the current layout and content of all notes on the canvas, ensuring no data loss.

## Style Guidelines:

- Light color scheme, conveying spaciousness and calm.
- Primary action color: A muted blue-grey (#606C80), subtle yet clear, for highlighting interactive elements or key information.
- Background color: An almost white, very light blue-grey (#F2F5F8), creating a clean and expansive 'slate' for the infinite canvas.
- Accent color: A soft, desaturated blue-cyan (#61ACC0), used sparingly for gentle emphasis or secondary highlights, harmonious with the primary.
- All text uses 'Inter' (sans-serif) for its modern, neutral, and highly readable quality, fitting the app's clean and minimalistic aesthetic.
- Use thin, minimalist line icons with consistent stroke weights, complementing the clean interface and focusing on functionality.
- Implement a free-form, gridless layout that allows notes to be placed anywhere on the infinite canvas, eschewing traditional boxes or columns.
- Incorporate subtle, fluid animations for note creation, movement, and canvas navigation (panning and zooming) to enhance the sense of continuous flow.
